
В данной директории находится программа-заглушка для sendmail,
предназначенная  для  отладки почтовых скриптов, а также отладочный
почтовый сервер, доступный по адресу localhost:25. В отличие от 
sendmail, система не посылает письма, а складывает их в директорию 
/tmp/!sendmail.

Файл sendmail.exe - эмулятор утилиты командной строки sendmail.
Файл sendmail_daemon.exe - почтовый сервер, эмулирующий прием почты.

Во врем работы Денвера sendmail_daemon.exe должен быть постоянно
запущен, чтобы скрипты имели возможность посылать письма по SMTP.

Для лучшей совместимости с Unix копия sendmail.exe также имеется в
директориях /usr/sbin/, /usr/bin/, /usr/local/sbin/ и /usr/local/bin/.
